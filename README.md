## 발표시간 계산기 (PT time Calculator)

> ### 소개

- 발표 스크립트를 넣으면, 발표시간을 예측해주는 웹사이트입니다.

---

> ### 주요기능

1. 사용자가 개인의 말하기 속도, 발표환경에 맞게 조절할 수 있습니다.
2. PPT 장수, PPT 사이의 쉼, 자료 감상 시간을 고려하여, 보다 정확하게 발표시간을 예측할 수 있습니다.
3. 글자수 세기, 단어수 세기 기능을 제공합니다.
4. 영어버전 지원
3. 글자수 세기, 단어수 세기 기능을 제공합니다.
